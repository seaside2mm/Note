

# debug 工具

  

- **nvcc**

`-archsm_86` : 为不同架构生成设备代码
`--device-debug (-G)` : 关闭大多数优化
`-lineinfo` : 文件名和行号关联

  

`-Xcompiler -rdynamic` : 堆栈信息
